### Автоматизована перевірка норм кодування програмного коду

У цьому розділі наведено файли сценаріїв GitHub Actions та фрагменти
результатів їх виконання для перевірки:

- версій програмного забезпечення та СКБД;
- відповідності SQL-схеми даних нормам кодування;
- відповідності програмного коду мовою Python нормам кодування.
